# Contact Book

Simple CLI contact book implemented in Python.

Usage
-
Run the program with:

```powershell
python contact_book.py
```

Options in the menu:
- Add contact: enter name and a 10-digit numeric phone number.
- View all contacts: list stored contacts.
- Search contact by name: case-insensitive search.
- Exit: quit the program.

The program stores contacts in-memory as a list of dictionaries, e.g. `{"name": "Alice", "phone": "1234567890"}`.
